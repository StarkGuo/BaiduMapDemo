# BaiduMapDemo
参考百度android地图文档写的迷你步行导航地图
# 能干什么？
搜素附近约100公里的路标，比如厕所、酒店、公交站
搜索完出现标签，点击标签弹出吐司显示标签具体信息，并显示步行的最短路径
# 注意什么？
每个key有对应的秘钥，如果自己用自己的key文件编译，要去百度申请秘钥，这样才能成功调用百度地图的api
获取秘钥具体方式在百度地图的开发指南。。。
